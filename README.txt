== Description ==

= Introduction =

Auto subscribe to popular email marketing tool like campaign monitor, constant contact, getresponse, icontact, infusionsoft, mailchimp for WooCommerce, Jigoshop, Fast Secure Contact Form, Contact form 7, Contact Form by BestWebSoft, WR ContactForm, Ninja Forms, Caldera Forms, JetPack Contact Form, Quform, Gravity Contact Forms and Enfold theme contact forms.

This useful little plugin currently supports addition of subscribers to all AutoResponders lists. The AutoResponder plugin allows you to automatically add people to your campaign monitor, constant contact, getresponse, icontact, infusionsoft, mailchimp list after they've sent you a message using your Contact form. You can either choose to automatically attempt to subscribe people to your list by simply enabling the "Enable Contact Form" in the Integration.

This plugin is used to sync the email address of WooCommerce and Jigoshop orders to Autoresponders and allows to automatically add customer to your Autoresponders lists after they've generate WooCommerce and Jigoshop Orders. they've add Email of Customers to particular WooCommerce and Jigoshop Orders status you can choose lists. Sync WooCommerce and Jigoshop Orders Email of Customers to connected account of all AutoResponder.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Autoresponder" and click Search Plugins. Once you've found our plugin you can view details about it such as the the rating and description. Most importantly, of course, you can install it by simply clicking Install Now.

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting previous versions if they exist.
2. Activate the plugin in your WordPress admin area.


= configuration =

Easy steps to install the plugin:

*	Upload "autoresponder" folder/directory to the /wp-content/plugins/ directory.
*	Activate the plugin through the 'Plugins' menu in WordPress.
